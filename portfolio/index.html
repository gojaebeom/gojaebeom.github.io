Intro

반갑습니다!
저는 꾸준히 지식을 탐구하는 웹 개발자 입니다.

저는 개발자로 취업을 준비 중인 고재범 입니다.
원래 전공은 아니었지만, 뒤늦게 웹 프로그래밍의 매력에 빠져 매일 공부하며 꾸준히 잠재력을 키워나가는 중입니다.

프로젝트를 만들거나 웹 관련 공부를 할 때 막히던 부분도 많이 있었습니다.
내가 할 수 있는건가? 생각하는 순간들도 포기하지 않으면 어느샌가 이해하고 있는 자신을 보는 게 너무 좋습니다.

기본기는 JAVA 언어로 시작하여 Spring 프레임워크, DB를 다루는 Back-end 분야를 주로 공부하고 있지만, 사용자에게 시각적으로 어필할 수 있는 Front-end 역시 관심을 가지고 알아가고 있습니다.

저는 웹 개발이 좋습니다. 잘하고 싶습니다. 그래서 많은 시간을 개발에 투자할 수 있고,
남들에게 인정받는 개발자가 되기 위해 더욱더 노력할 것입니다.
나를 대표하는 키워드
열정 끈기 덕후 감성

Skill

제가 잘하는 것들을 소개합니다.



🔥 웹 퍼블리싱(Web Publishing)

html, css를 이용하여 기본적인 레이아웃을 구성하고 javascript를 활용하여 동적인 효과를 줄 수 있습니다.

🔥 자바 웹 프로그래밍(Java Web Programming)

자바의 문법과 객체지향 프로그래밍의 특징을 잘 이해하고 있습니다.
자바 웹 프로그래밍을 위한 서블릿 클래스를 상속받아 웹 프로젝트를 구성할 수 있습니다.
서블릿과 jsp의 실행 구조를 이해하고 있습니다.

🔥 스프링 프레임워크(Spring Framework)

maven 빌드툴을 활용하여 spring-mvc 기반의 웹 프로젝트를 만들 수 있습니다.
spring이 제공하는 ioc 컨테이너를 적극적으로 활용하여
controller, service, dao 객체간의 결합을 느슨하게 조합하여 프로그래밍 합니다.
아직 사용하지 못한 기능들도 많아 프로젝트를 만들며 계속 공부중에 있습니다.

🔥 관계형 데이터베이스(MySQL)

프로젝트에 필요한 구조의 테이블을 생성하고 기본적인 쿼리문을 사용할 수 있습니다.

🔥 자바스크립트(Javascript)

기본적인 문법과 es6 이상에서 추가되는 기술들을 공부중에 있습니다.
필요한 부분에 ajax 를 통한 비동기 통신을 사용할 수 있습니다.
React, nodejs 를 이용하여 (간단한)프로젝트를 만들 수 있습니다.

Project

제가 만든 작품들 입니다.

방송대 QA

스프링과 jsp를 이용하여 만든 웹 사이트 입니다. 방송대 학우들이 소통할 수 있는 질문, 답변형 커뮤니티 사이트입니다.
CRUD 이메일 인증 비밀번호 암호화 비밀번호 찾기 댓글 이미지 업로드 태그 등록 좋아요 기능 조회수 처리 페이징 처리 게시물, 회원, 태그 검색

Storago

React와 NodeJs(Express)로 만든 심플한 웹사이트 입니다. 만들어둔 프로젝트들을 전시하는 웹 사이트 입니다. 만든 파일들을 올리고 다운받을 수 있습니다.
CRUD 관리자 로그인 파일 업로드, 다운로드
